бүтээсэн. Энэ систем нь:

1. ✅ MCP (Model Context Protocol) интеграци
2. ✅ PostgreSQL database холболт
3. ✅ Байгалийн хэл дээрх асуултуудыг ойлгох
4. ✅ Борлуулалтын мэдээлэл авах (SKU, merchant, district, цаг хугацаа, sales rep)
5. ✅ Insights үүсгэх (top SKU, trends, patterns)
6. ✅ Web UI (chat interface)
7. ✅ RESTful API

Backend:
---------

- run.py               : Серверийг ажиллуулах

Frontend:
---------
- static/index.html    : Chat UI HTML
- static/style.css     : CSS стил
- static/script.js     : JavaScript (API холболт)

Utilities:
----------
- requirements.txt     : Python dependencies
- mcp_config.json      : MCP тохиргоо

API ENDPOINTS
================================================================================

POST /api/query
   Байгалийн хэл дээрх асуултанд хариулах
   Body: {"question": "Хамгийн их борлуулалттай SKU-ууд юу вэ?"}

POST /api/sales/sku
   SKU-аар борлуулалт

POST /api/sales/merchant
   Merchant-аар борлуулалт

POST /api/sales/district
   District-аар борлуулалт

POST /api/sales/time-period
   Цаг хугацааны дагуу борлуулалт

POST /api/sales/sales-rep
   Sales rep гүйцэтгэл

GET /api/insights/top-skus
   Top SKU-ууд

GET /api/insights/district-trends
   District trends

GET /api/insights/category-summary
   Category summary

GET /api/insights/merchant-patterns
   Merchant ordering patterns

ТЕХНИКИЙН ДЭЛГЭРЭНГҮЙ
================================================================================

Tech Stack:
-----------
- Backend: Python 3.8+, FastAPI, Uvicorn
- Database: PostgreSQL (psycopg2)
- AI: Google Gemini Pro, google-generativeai
- Frontend: HTML, CSS, JavaScript (Vanilla)
- MCP: Model Context Protocol

Database Schema (Хүлээгдэж буй):
---------------------------------
- merchants (id, name, district)
- sku (id, name, category)
- orders (id, merchant_id, sales_rep_id, order_date)
- order_items (id, order_id, sku_id, quantity, price)
- sales_reps (id, name)
